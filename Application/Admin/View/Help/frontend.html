编码规范

1.缩进对齐
	使用tab缩进
	多行变量赋值,数组赋值,使用缩进和空格来对其, 如：
		var name      = '';
		var yourName  = '';
		var arr       = new Array = [
			'name',
			'age',
			'birthday',
		];
	
	
2.变量
	变量名称使用有意义的英文字母  如: userPoint
	普通的变量采用驼峰, 如 myName
	
	
3.控制结构 if,for,foreach,while,switch
	注意多个条件之间的空格 和 括号的位置 如:
	if((con1 && con2) || con3){
	}
	
4.函数名
	采用驼峰
	
	
5.嵌套深度
	避免深层嵌套,深度不要超过3级
	
6.注释
	添加应有的注释,函数和类必须有注释
	当行注释 ,如://这是一个单行注释
	多行注释, 如:(注意以"/**" 开始, "*/"结尾)
		/**
		 *
		 *
		 */

项目规范

1.常用或基础函数写在一个js文件内

2.常用ajax操作代码复用 
	“编辑添加” ajaxEdit
	“删除”     ajaxDel

4.现的功能
	时间选择控件 		在时间选择input标签中添加一个`laydate` class属性，如果需要精确到时间，请在input标签中添加`data-istime="true"`属性
							
	radio			select/radio
	方法及参数			radio($data)
						{	
						* @param array $data 选择按钮的信息
						举例:$data = ['list'=> ['男'=>'男','女'=>'女'], 'name'=>'gender', 'checked'=>$row['gender']]
							{
							* @param array $data['list'] 是模型里取的信息
							* @param string 'name'是字段名，同时也是radio的name值
							* @param 'checked' 是radio的value值，是从list数组中获取的键值
							}
						}
						
	select			select/select
	方法及参数			select($data)
						{	
						* @param array $data 选择框的信息
						举例:$data = [[ 'name'=>'type', 'list' => $mod->typeArr]]
							{
							* @param array $data['list'] 是模型里定义好或者取数据库的信息
							* @param string 'name'是字段名，同时也是select的name值
							* @param option的value值是从list数组中获取的键值
							}
						}
						
	checkbox		select/checkbox
	
	
	方法及参数			checkbox($data)
						{	
						* @param array $data 选择按钮的信息
						* @param array $data['list'] 是模型里定义好或者取数据库的信息
						* value值是从list数组中获取
						* list的结构是:[['name'=>'', 'value'=>'value',checked=>true],['']]
						}
						
						
	
	富文本框		edit/index 在textarea 的class属性添加"kind-editor"
	日期选择控件  	在input 的class属性添加“lay-data”
